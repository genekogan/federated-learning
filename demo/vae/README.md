# Federated VAE (work in progress)


```
npm install -g yalc
cd demo/vae/server
yarn
```

Install client deps:
```
cd demo/vae/client
yarn
```

## Running the Demo

To start the server first time or in dev mode:
```
cd demo/vae/server
yarn dev
```

To just run it:

```
yarn go
```


To start the client:
```
cd demo/vae/client
yarn watch
```
